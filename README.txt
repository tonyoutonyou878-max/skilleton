SmartCampusPortal

Setup instructions:
1. Install PHP and MySQL on your system.
2. Place the SmartCampusPortal folder in your web server root (e.g., htdocs or www).
3. Create a MySQL database named 'cusit_portal'.
4. Import cusit_portal.sql into the database.
5. Update DB_USER and DB_PASS in config.php if needed.
6. Open http://localhost/SmartCampusPortal/index.php in your browser.
